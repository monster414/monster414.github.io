[2018强网杯]Streamgame1

加密流程

  1. 序列左移一位
  2. 与反馈系数进行且运算
  3. 运算结果进行异或得到结果
  4. 结果作为序列的结尾

result=cn&an⊕...⊕c2&a2⊕c1&a1result = c_n\&a_n \oplus ... \oplus c_2\&a_2 \oplus c_1\&a_1result=cn​&an​⊕...⊕c2​&a2​⊕c1​&a1​
反馈系数为 mask = 0b1010011000100011100
即 c19,c17,c14,c13,c9,c5,c4,c3=1c_{19}, c_{17}, c_{14}, c_{13}, c_9, c_5, c_4, c_3 = 1c19​,c17​,c14​,c13​,c9​,c5​,c4​,c3​=1 (从右到左)
那么反馈运算为
result=a19⊕a17⊕a14⊕a13⊕a9⊕a5⊕a4⊕a3result = a_{19} \oplus a_{17} \oplus a_{14} \oplus a_{13} \oplus a_9 \oplus a_5 \oplus a_4 \oplus a_3result=a19​⊕a17​⊕a14​⊕a13​⊕a9​⊕a5​⊕a4​⊕a3​

而且我们已知a1-a18以及产生的result
那么就可以根据这些条件来得出a19
a19=result⊕a17⊕a14⊕a13⊕a9⊕a5⊕a4⊕a3a_{19} = result \oplus a_{17} \oplus a_{14} \oplus a_{13} \oplus a_9 \oplus a_5 \oplus a_4 \oplus a_3a19​=result⊕a17​⊕a14​⊕a13​⊕a9​⊕a5​⊕a4​⊕a3​
以此类推，来恢复flag
